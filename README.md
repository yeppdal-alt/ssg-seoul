# 🏪 서울시 상가(상권) 정보 분석 대시보드

소상공인시장진흥공단이 제공하는 **서울시 상가(상권)정보(2026년 3월 기준)** 를 기반으로,
지역별 · 업종별 상권 현황을 인터랙티브하게 탐색할 수 있는 Streamlit 대시보드입니다.

## 데모 미리보기

- 상단: 강남구 · 서초구 · 마포구 · 용산구의 업종 TOP5를 한눈에 비교
- 하단: 원하는 지역을 선택하거나 직접 입력해 해당 상권의 업종 현황을 그래프로 확인

## 주요 기능

- **주요 4개 구 TOP5 대시보드**: 강남구, 서초구, 마포구, 용산구의 업종 대분류 TOP5를 최상단에 고정 표시
- **지역 검색 (선택 + 직접 입력)**: 구 단위 드롭다운 선택 또는 `✏️ 직접 입력`으로 구/행정동/법정동 이름 일부만 입력해도 검색 가능, 행정동 단위로 추가 필터링 가능
- **업종 현황 시각화 (Plotly)**: 업종 대분류 막대·파이 차트, 업종 중분류 TOP15, 대분류→중분류 트리맵
- **위치 지도**: 선택한 지역 업소들의 위경도 기반 산점도 지도 (업종별 색상 구분)
- **KPI 카드**: 선택 지역의 전체 업소 수, 업종 다양성, 최다 업종 등 핵심 지표 요약


## 데이터 설명

- **원본**: 소상공인시장진흥공단 상가(상권)정보 - 서울 (2026년 3월 기준), 537,489개 업소 / 39개 컬럼
- **가공**: GitHub 업로드 용량 제한(20MB)에 맞추기 위해 다음 두 단계로 축소했습니다.
  1. 분석에 필요한 9개 컬럼만 추출: `상호명`, `상권업종대분류명`, `상권업종중분류명`, `상권업종소분류명`, `시군구명`, `행정동명`, `법정동명`, `경도`, `위도`
  2. 구(시군구)별 비율을 유지한 **30% 층화 샘플링** 적용 → 161,247건, 18.9MB
- **검증**: 강남구 · 서초구 · 마포구 · 용산구의 업종 TOP5 순위가 원본 전체 데이터와 샘플 데이터에서 동일함을 확인했습니다.
- 원본 전체 데이터(약 290MB)로 더 정밀한 분석이 필요하다면 `data_seoul.csv`를 원본 CSV로 교체하고 `app.py`의 `DATA_PATH`만 수정하면 됩니다.

## 실행 방법

### 로컬 실행

```bash
git clone <이 저장소 URL>
cd <저장소 폴더>
pip install -r requirements.txt
streamlit run app.py
```

### Streamlit Community Cloud 배포

1. 이 저장소를 GitHub에 push (app.py, requirements.txt, data_seoul.csv가 같은 폴더에 있어야 함)
2. [share.streamlit.io](https://share.streamlit.io) 에서 New app 생성
3. 저장소 선택 → Main file path에 `app.py` 지정 → Deploy

## 기술 스택

- [Streamlit](https://streamlit.io/) — 웹 앱 프레임워크
- [Plotly](https://plotly.com/python/) — 인터랙티브 시각화 (막대, 파이, 트리맵, 지도)
- [Pandas](https://pandas.pydata.org/) — 데이터 처리
